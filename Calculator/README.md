# Calculator

## Overview

This project is a Basic Calculator developed using HTML, CSS, and JavaScript as part of the CODSOFT Web Development Internship (Level 1 - Task 3).

The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division through an interactive user interface.

## Features

* User-friendly calculator interface
* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Clear button (C)
* Backspace functionality
* Real-time display screen
* Responsive design
* Error handling for invalid expressions

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Functionalities

### Display Screen

Shows the entered numbers, operators, and calculated results.

### Arithmetic Operations

Supports:

* Addition
* Subtraction
* Multiplication
* Division

### Clear Function

Removes all entered values and resets the calculator.

### Backspace Function

Deletes the last entered character.

### Result Calculation

Evaluates mathematical expressions and displays the result.

## How to Run

1. Download or clone the repository.
2. Open the Calculator folder.
3. Ensure all files are in the same directory:

   * index.html
   * style.css
   * script.js
4. Open `index.html` in a web browser.
5. Alternatively, use Live Server in VS Code.
