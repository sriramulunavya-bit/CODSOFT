# Landing Page

## Overview

This project is a responsive Landing Page developed using HTML and CSS as part of the CODSOFT Web Development Internship (Level 1 - Task 2).

The landing page is designed to demonstrate the fundamentals of web development, including page layout, navigation, sections, responsive design, styling, and user-friendly interfaces.

## Features

* Responsive design for desktop and mobile devices
* Navigation bar with menu links
* Hero section with call-to-action button
* Features section with information cards
* Services section
* About Us section
* Contact section
* Footer section
* Modern UI using CSS Flexbox and responsive layouts

## Technologies Used

* HTML5
* CSS3

## Sections Included

### Header

Contains the website logo/title and navigation menu.

### Hero Section

Displays the main heading, description, and call-to-action button.

### Features Section

Highlights the key features of the website using cards.

### Services Section

Showcases the services offered.

### About Section

Provides information about the organization or business.

### Contact Section

Contains contact information for users.

### Footer

Displays copyright information and additional details.

## Learning Outcomes

Through this project, the following concepts were practiced:

* HTML page structure

* Flexbox design

* Navigation menus

## How to Run

1. Download or clone the repository.
2. Open the Landing_Page folder.
3. Open `index.html` in any modern web browser.
4. Alternatively, use the Live Server extension in VS Code.
