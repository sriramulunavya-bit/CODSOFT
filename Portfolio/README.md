# Personal Portfolio Website

## Overview

This is a personal portfolio website developed using HTML and CSS as part of the CODSOFT Web Development Internship (Level 1 - Task 1).

The portfolio showcases personal information, skills, projects, resume, and contact details in a responsive and user-friendly design.

## Features

* Responsive website design
* Header section with name and introduction
* About Me section
* Skills section
* Projects showcase section
* Resume download option
* Contact information section
* Footer with copyright information

## Technologies Used

* HTML5
* CSS3

## Project Structure

Portfolio/

├── index.html

├── style.css

├── profile.jpg

├── resume.pdf

└── README.md

## Sections Included

### Header

Displays the name and a short professional tagline.

### About Me

Provides a brief introduction and profile picture.

### Skills

Lists technical skills and areas of expertise.

### Projects

Displays project titles and descriptions.

### Resume

Allows visitors to download the resume in PDF format.

### Contact

Contains email address, phone number, and location.

### Footer

Contains copyright information.

## How to Run

1. Download or clone the repository.
2. Open the Portfolio folder.
3. Double-click `index.html` or run it using Live Server in VS Code.
4. The portfolio website will open in your browser.
